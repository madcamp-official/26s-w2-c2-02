# Changelog

Roomi의 주요 변경 사항은 사용자 영향부터 적고, 내부 구현 세부사항은 그 뒤에 정리합니다.
루트 `README.md`는 안정적인 프로젝트 개요로 유지합니다.

## Unreleased

### Added

- 시선 이탈 감지를 추가했습니다. 고개는 화면을 향하는데 눈만 옆을 보는 상태가 5초 이상 이어지면 집중 안 함으로 판정합니다. MediaPipe 홍채 랜드마크로 눈이 눈구석 안에서 얼마나 치우쳤는지를 재서, 고개가 보는 방향과 시선이 보는 방향의 차이를 각도로 계산합니다. 읽는 동안 눈은 계속 움직이므로 각도(기본 30°)보다 지속 시간에 무게를 뒀습니다.
- 실시간 집중 순위 카드에 `자세히 보기`를 추가했습니다. 하품 빈도, 눈 깜빡임 빈도, 눈 감김 시간 비율에서 계산한 피로도와, 시선 이탈·고개 돌림·자리 비움 횟수·자세 흔들림에서 계산한 산만함을 보여줍니다. 피로도가 높으면 휴식을 권하고, 산만함이 높게 유지되면 집중 점수가 내려가는 `집중 안 함` 상태로 반영됩니다. 세션 1분이 지나야 표시됩니다.

### Changed

- 고개 돌림과 자세 흔들림의 인식률을 높였습니다. 고개 돌림은 3초, 고개 숙임은 5초 이상 이어지면 집중 안 함 후보로 잡히고, 고개 돌림 각도는 30°에서 28°로 조정했습니다. 각도를 더 내리지 않은 것은 실제 사용자를 측정한 편안한 범위가 ±25°여서, 그 아래로 가면 평소 앉아서 읽는 자세가 딴짓으로 잡히기 때문입니다. 자세 흔들림은 가만히 있는 머리의 노이즈 수준(0.3)을 바닥으로 삼아, 그 이상 움직이면 바로 반영됩니다.
- 공부 모드 화상회의 화면에 피로도 기반 `휴식 제안`과 산만함 기반 `집중 확인` 카드를 추가했습니다. 산만함이 높거나 고개/시선 신호가 집중 안 함으로 잡히면 점수가 내려가고, 사용자가 `집중 중이야`를 누르면 즉시 집중 상태로 복귀하며 방금 관측된 고개·시선·산만함 기준 근처는 더 넓게 허용합니다.
- 하품 판정 기준을 입이 1.5초 이상 벌어진 상태에서 1초로 줄였습니다. 짧은 하품도 잡힙니다. 말하다가 하품으로 오인되는 것은 주로 입 벌림 비율(0.6) 기준이 막고 있어서, 지속 시간은 말하는 중의 순간적인 입 벌림만 넘기면 됩니다.
- 화상 타일에서 집중 중인 참여자를 `참여 중` 대신 `공부 중`으로 표시합니다. 표정 파티 게임방에서는 같은 자리에 `게임 중`이 나옵니다.
- 대기실 `함께하는 사람들` 목록에서 이름 첫 글자 아바타를 없애고 이름만 보여줍니다.

- 화상 회의방의 나가기 버튼이 대기실 대신 회고 화면으로 이동하도록 바꿨습니다. 방장이 세션을 종료하면 방 상태 갱신을 받은 참여자도 회고 화면으로 이동합니다.
- 게임 모드 회고 화면을 게임 결과용 레이아웃으로 분리해 최종 순위, 내 점수, 우승자, 라운드별 결과와 미션 진행 정보를 보여주도록 했습니다.
- 숨은 표정 미션의 방해 카드가 개별 웹캠 타일이 아니라 전체 참가자 웹캠 그리드 위를 덮도록 조정했습니다.
- 방 만들기 화면에서 카피캣 릴레이를 게임 2로 올리고, 게임 3 포커페이스 블러프는 다음 업데이트 안내만 표시되며 선택되지 않도록 바꿨습니다.
- 숨은 표정 미션의 윙크 판정을 더 엄격하게 바꿨습니다. 한쪽 눈을 감는 동안 반대쪽 눈이 확실히 열려 있어야 윙크로 인정됩니다.
- 숨은 표정 미션의 모든 성공 필요 횟수를 2회씩 늘려, 각 미션이 이전보다 조금 더 오래 이어지도록 조정했습니다.
- 숨은 표정 미션에 지목 기능을 추가했습니다. 누군가 미션을 수행한 직후 1.5초 안에 수행자를 지목하고 미션을 맞히면 지목자가 라운드 승자가 되며, 오답 후에는 10초 동안 다시 지목할 수 없습니다.
- 숨은 표정 미션의 방해 카드가 게임 시작 10초 이후부터 30초 안의 랜덤한 시점에 나타나도록 바꿨습니다. 방해 카드를 풀면 다시 10~30초 뒤 새 방해 카드가 예약됩니다.
- 숨은 표정 미션의 방해 카드가 1인 플레이에서도 나오도록 바꿨습니다.

- 라운드 사이 상단 필드에 표시되는 직전 라운드 우승자 문구를 카운트다운보다 작은 크기로 분리해, 다음 라운드 시작 시간이 더 또렷하게 보이도록 조정했습니다.
- 숨은 표정 미션의 윙크와 고개 끄덕이기 감지 민감도를 높였습니다. 작은 윙크와 가벼운 고개 끄덕임도 더 잘 카운트되도록 기준을 낮췄습니다.
- 숨은 표정 미션 라운드 점수를 성공 여부만 보지 않고 진행 비율에 따라 0~10점으로 부여하도록 바꿨습니다. 라운드 사이 상단 필드에는 방금 끝난 라운드의 우승자도 함께 표시합니다.
- 라운드 사이 `현재 순위` 카드에서 참가자 이름과 `준비 대기 중`/`준비 완료` 상태가 더 떨어져 보이도록 간격을 조정했습니다.
- 숨은 표정 미션 시작 멘트가 라운드별 대화 주제를 먼저 꺼내도록 바꿨습니다. 참가자는 그 주제로 대화하면서 비밀 미션을 자연스럽게 섞을 수 있고, 게임방 채팅에 새 메시지가 올라오면 루미가 최근 채팅만 참고해 한국어로 짧게 반응합니다.
- 숨은 표정 미션 중 루미의 중간 경고는 아무 카운트나 오를 때가 아니라 누군가 목표까지 1회 남았을 때만 `누군가 거의 미션을 끝낸 것 같은데...?`로 고정해 표시합니다.
- 숨은 표정 미션 라운드에 `순간 기억`, `빠른 선택`, `다른 표정 찾기` 방해 카드를 추가했습니다. 카드는 화상 타일을 가리지만 접었다 펼 수 있고, 풀기 전까지 본인의 미션 카운트가 잠깁니다.
- 숨은 표정 미션에서 `입 크게 벌리지 않기` 방어형 미션을 제거하고, 입 살짝 벌리기와 고개 끄덕이기 미션을 추가해 후보를 14개로 확장했습니다.
- 스터디룸 화면을 표정 파티 게임 중심 경험으로 전환했습니다. 사용자는 숨은 표정 미션, 점수판, Roomi 진행 메시지, 참가자 비디오 타일을 한 화면에서 확인할 수 있습니다.
- 중앙 API가 일시적으로 응답하지 않는 개발 환경에서도 데스크톱 앱이 로컬 데모 방을 만들어 표정 게임 UI와 로컬 카메라 흐름을 실행할 수 있게 했습니다. 다중 PC 초대, Daily 토큰, Socket.IO 동기화, 서버 권위 점수 계산은 여전히 중앙 API가 필요합니다.
- 공유 타입과 Socket.IO 계약에 `hidden_mission`, `poker_bluff`, `copycat_relay` 게임 상태, 비공개 미션 배정, 표정 결과 보고, 게임 공개 이벤트를 추가했습니다.
- API 서버가 게임 시작, 미션 결과 기록, 블러프 베팅/결과, 릴레이 진행, 공개 시점 점수 계산을 처리하도록 확장했습니다. 공개 전 숨은 미션은 대상 참가자에게만 전달됩니다.
- 데스크톱 렌더러가 MediaPipe blendshape 결과를 표정 신호로 변환하고, 숨은 미션 진행도를 로컬에서 계산한 뒤 서버 또는 로컬 데모 상태에 보고하도록 연결했습니다.
- 데스크톱 방 만들기 화면에서 `공부하기`, `hidden_mission`, `poker_bluff`, `copycat_relay` 중 하나를 고르게 했습니다. 휴식 설정과 휴식 버튼은 공부하기 모드에서만 보이고, 게임 모드는 방 생성 때 정한 게임으로 라운드를 시작합니다.
- 게임 모드 대기실의 목표 입력 영역을 `오늘의 플레이 스타일` 설정으로 바꾸고, 루미가 게임별 스타일을 추천해 저장한 뒤 게임 중 진행 멘트에서 참고할 수 있게 했습니다.
- 게임 모드 대기실의 긴 제목이 초대 코드를 밀지 않도록 더 작게 줄이고, 진행 중인 게임방 참여자는 `게임 중`으로 표시되게 했습니다. 방 만들기 화면의 방 방식 선택은 여유 있는 가로 스크롤 카드 레일로 바꿨습니다.
- 화상회의 화면에서 모든 참가자의 공부 목표 또는 플레이 스타일을 볼 수 있게 했고, 초대 코드·진행자·플레이 스타일·게임 조작이 있는 오른쪽 패널만 별도로 스크롤되게 했습니다. 숨은 표정 미션은 성공 전 중간 카운트가 올라가도 루미가 반응하도록 했습니다.
- 숨은 표정 미션 카운트에 짧은 쿨다운을 추가해 눈썹 올리기 같은 빠른 표정 변화가 과하게 연속 집계되지 않게 했고, 루미의 미션 반응은 참가자 이름이나 성공 여부를 직접 말하지 않고 보이는 표정 단서만 넌지시 암시하도록 바꿨습니다.
- 눈썹 올리기 미션은 더 높은 감지 기준과 낮은 해제 기준을 따로 적용해, 눈썹이 계속 보이거나 임계값 근처에서 흔들릴 때 추가 카운트가 오르지 않게 했습니다.
- 화상회의 화면의 오른쪽 패널 카드들이 화면 높이에 따라 압축되지 않게 해 `루미 진행자` 메시지 창이 루미 메시지 길이에 맞춰 유지되도록 했습니다.
- 로딩 페이지 캐치프레이즈를 `친구들과 함께 켜두는 AI 플레이룸`으로 바꿔 공부와 게임을 모두 담는 방향으로 조정했습니다.
- 방 만들기 화면의 `방 방식` 카드 레일에 여백을 더하고, 숨은 표정 미션 후보에서 `생각하는 척하며 볼을 3번 부풀리기`를 제외했습니다.
- 게임 모드 입장 직후 루미 기본 메시지가 공부 시작 멘트 대신 선택된 게임에 맞는 안내로 나오도록 바꿨습니다.
- 숨은 표정 미션이 라운드마다 셔플된 여러 미션 후보에서 배정되도록 늘렸고, 게임 시작·플레이어 반응·공개 시점마다 루미가 한국어 LLM 또는 한국어 템플릿 fallback으로 실시간 진행 멘트를 보내도록 했습니다.
- 게임 모드 방 만들기에서 세션 시간 대신 라운드 수를 정하도록 바꾸고, 게임방 상단에는 현재 라운드 또는 다음 라운드 시작 카운트다운을 표시하도록 했습니다. 라운드 사이에는 참가자별 `다음 라운드 준비` 상태를 모아 모두 준비하면 즉시, 5분이 지나면 자동으로 다음 라운드가 시작됩니다.
- 게임방의 플레이 스타일 카드를 `현재 순위` 카드로 바꾸고, 자세히 보기 모달에서 종합 순위, 라운드별 결과, 다음 라운드 준비 여부, 각자의 플레이 스타일을 함께 확인할 수 있게 했습니다. 숨은 표정 미션은 성공자가 나오면 라운드를 멈추고 결과 확인 버튼을 보여줍니다.
- 숨은 표정 미션에서 참가자가 대기실에 갔다가 다시 게임방에 들어오면 같은 라운드 안에서도 개인 미션을 새로 배정하도록 했습니다.
- 화상회의 타일의 상태 레이블을 MediaPipe 신호에 맞춰 `얼굴 없음`, `눈 감김`, `시선 이탈`, `고개 숙임`, `집중 흔들림` 등으로 더 자세히 표시하고, 게임방에서는 얼굴이 감지되지 않는 즉시 `자리 비움` 상태로 전환되도록 했습니다. 공부 모드는 기본값인 5초를 유지합니다.
- 화상회의 화면의 작은 `화상 연결` 상태 텍스트와 별도 `게임 결과 확인` 배너를 제거해, 결과 확인은 `현재 순위` 카드의 `자세히 보기`로만 열리게 정리했습니다.
- `docs/api.md`와 `docs/architecture.md`를 표정 파티 게임 이벤트 계약과 중앙 API 장애 시 로컬 개발 fallback에 맞춰 갱신했습니다.
- 숨은 표정 미션의 윙크와 눈썹 올리기 감지 기준을 조금 낮춰, 과한 동작 없이도 카운트가 오르도록 민감도를 높였습니다.
- 집중 순위 기준을 집중 시간에서 집중 점수로 바꿨습니다. 점수는 집중한 5초마다 10점씩 오르고, 흐트러진 동안 5점씩, 자리를 비운 동안 10점씩 깎이며 0점 아래로는 내려가지 않습니다. 휴식과 눈 감김 상태에서는 점수가 오르지도 깎이지도 않습니다. 오래 자리를 비워도 마이너스가 쌓이지 않아서, 돌아오면 곧바로 다시 점수를 쌓기 시작합니다.
- 화상 타일의 상태 표시를 두 칸으로 나눴습니다. 왼쪽은 지금 카메라가 보고 있는 것(`고개 숙임`, `고개 돌림`, `입 벌림`, `하품`, `눈 감김`, `얼굴 없음`), 오른쪽은 그것이 집중으로 인정되는지 여부(`집중중`, `집중 안 함`, `휴식`, `대기`)입니다. 무슨 일이 일어나는지와 그게 집중인지가 한 레이블에 섞여 있던 것을 분리했습니다.
- 고개 숙임과 고개 돌림 레이블이 지속 시간을 기다리지 않고 그 순간 바로 바뀝니다. 집중 점수와 순위 판정은 기존 지속 기준(10초)을 그대로 써서 영향이 없습니다. 눈 감김과 하품은 지속 기준을 유지합니다. 깜빡임마다 `눈 감김`이 번쩍이거나 말하는 중에 `하품`이 뜨면 레이블이 사실과 달라지기 때문입니다.
- 상세 레이블은 지금처럼 본인 타일에만 보입니다. 다른 참가자 타일에는 집중 여부만 표시되고 구체적인 이탈 사유는 공개되지 않습니다.
- 화상회의 화면의 `실시간 집중 순위` 카드에 내 점수와 최근 점수 흐름 그래프를 추가했습니다. 집중하면 선이 올라가고 흐트러지거나 자리를 비우면 내려가서, 점수가 깎이는 순간을 바로 볼 수 있습니다.
- 공부 모드에서 1분 이상 집중이 흐트러진 참가자에게도 루미가 본인에게만 확인 메시지를 보냅니다. 지금까지는 얼굴이 사라진 `자리 비움`에만 반응했습니다. 감지는 틀릴 수 있으므로 메시지는 딴짓한다고 단정하지 않고 아직 목표 작업 중인지 먼저 묻고, 참가자가 스스로 집중을 되찾으면 아무 말도 하지 않습니다. 반복 알림을 막기 위해 흐트러짐 확인 메시지는 10분에 한 번까지만 보냅니다.
- 공부 모드에 하품 감지를 추가했습니다. 입이 크게 벌어진 상태가 1.5초 이상 이어지면 하품으로 보고 집중 점수를 15점 깎으며, 화상 타일에 `하품`으로 표시합니다. 말하기나 웃음은 입이 짧게만 벌어져서 하품으로 잡히지 않습니다. 하품 한 번으로 상태가 `자리 비움`이나 일시정지로 바뀌지는 않습니다. 졸린 것과 공부를 안 하는 것은 다르기 때문입니다.
- 판정 테스트 화면에 깜빡임 빈도(회/분)와 하품 횟수를 표시합니다. 두 값은 아직 집중 판정에 쓰지 않습니다. 평상시 깜빡임 횟수는 사람마다 편차가 커서 고정 기준을 지금 정하면 오탐이 늘어나기 때문이고, 개인별 기준값을 잡은 뒤에 판정에 넣을 예정입니다.
- 고개 돌림·숙임 판정이 랜드마크 비율 근사 대신 MediaPipe 6DoF 행렬에서 계산한 실제 각도(도)를 씁니다. 얼굴이 화면 가장자리에 있거나 카메라에 가까워도 값이 흔들리지 않고, 임계값을 사람이 읽을 수 있는 단위로 다룰 수 있습니다. 기본값은 편하게 앉아 움직이는 범위(yaw ±25°, pitch +20° 안팎) 바깥인 고개 돌림 30°, 고개 숙임 25°로 잡아 평소 자세와 필기가 흐트러짐으로 잡히지 않게 했습니다. 판정 테스트 화면의 임계값 슬라이더도 각도 단위로 바뀌고, 실제 각도를 함께 표시합니다.

### Fixed

- 실시간 집중 순위가 세션 내내 멈춰 있던 문제를 수정했습니다. 순위는 상태가 바뀔 때만 값을 갱신했는데, 계속 집중하고 있으면 상태 변화가 일어나지 않아서 10분을 집중해도 `0분`이 그대로 방송됐습니다. 12초 heartbeat가 있어도 같은 멈춘 값을 다시 보낼 뿐이었습니다. 이제 마지막 상태 변화 이후 흐른 시간까지 반영해 실시간으로 올라갑니다.
- 공부 모드에서 얼굴이 잠깐만 안 잡혀도 곧바로 `자리 비움`이 되던 문제를 수정했습니다. 얼굴을 가리는 것 자체가 플레이인 게임방용 설정이 공부 모드에도 함께 걸려 있었습니다. 물을 마시거나 손이 얼굴을 스치는 순간마다 자리 비움으로 바뀌었고, 그동안 집중 시간이 쌓이지 않아 회고와 랭킹 숫자가 깎였습니다. 공부 모드는 다시 기본값인 5초를 적용하고, 게임방은 지금처럼 즉시 전환합니다.
- 채팅에 반응하는 루미 메시지가 `닉네임: 원문` 또는 사용자 채팅 원문을 앞에 반복해서 붙이던 문제를 후처리로 제거했습니다.
- 숨은 표정 미션에서 새 라운드가 시작되는 순간 이전 라운드의 성공 카운터가 새 미션으로 잘못 제출되어 2라운드가 건너뛰는 문제를 수정했습니다. 로컬 fallback에서도 라운드가 바뀔 때 직전 미션과 같은 문장이 다시 배정되지 않도록 했습니다.
- 채팅에 반응하는 루미 메시지가 사용자의 채팅을 따라 하거나 요약하는 대신, 채팅 내용에 대한 짧은 답변이나 구체적인 후속 질문을 하도록 프롬프트와 fallback 문구를 조정했습니다.
